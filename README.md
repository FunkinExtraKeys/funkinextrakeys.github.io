# funkinextrakeys.github.io
GitHub Pages site you can use to redirect players quickly to Extra Keys download links.

URLs:

`https://funkinextrakeys.github.io/{engine}`
  - `/{platform}`
  - `/gb/`
  - `/repo/` (GameBanana mod URL)
  - `/` (Redirect to source code)

Fill-ins:

- `{engine}`:
  - `vslice` V-Slice
  - `psych` Psych Engine
- `{platform}`:
  - `windows` Windows
  - `mac` MacOS
  - `linux` Linux
  - `android` Android
